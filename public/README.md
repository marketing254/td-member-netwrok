# Thriving Dentist — Member Network
## Logo asset pack

**Primary lockup:** V2 horizontal · light

### Files in `exports/`
- `td-logo-horizontal-light.svg` — primary lockup, light backgrounds
- `td-logo-horizontal-dark.svg`  — primary lockup, dark backgrounds
- `td-sigil-mark.svg`            — icon only (avatar, favicon source)
- `td-sigil-mark-dark.svg`       — icon only on dark
- `td-sigil-mark-mono.svg`       — single-color (print, embossing, fax)

### Tokens
| Role | Hex |
|---|---|
| Ink (primary text/mark) | `#1a1f24` |
| Ivory (light bg, knockout) | `#f6f1e7` |
| Brass (accent dot) | `#b5894a` |
| Teal Deep (alt brand bg) | `#16363a` |

### Type
- Wordmark: **Inter Tight** 600, letter-spacing −0.02em
- Subline: **Inter Tight** 400, letter-spacing 0.22em, uppercase

### Clearspace
Minimum clearspace = height of the "T" cap on all sides.

### Min size
- Horizontal lockup: 120px wide minimum on screen.
- Mark alone: 16px minimum (favicon).

### Don'ts
- Don't recolor the brass dot to non-accent hues.
- Don't stretch or rotate the lockup.
- Don't place the light variant on busy or low-contrast imagery — use the dark variant or add a solid plate.
